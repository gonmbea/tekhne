# TEKHNE

**TEKHNE** is a personal web project exploring the intersection of technology, and art.  
The name *Tekhne* (from the ancient Greek **τέχνη**) means *art*, *craft*, or *skill* , a term that bridges both creative and technical disciplines. I chose the name because it represents the idea of crafting something thoughtful and intentional, where technology itself becomes a form of art.

The user interface was inspired by [Meowni](https://meowni.ca), but the entire site was built independently using mostly **HTML** and **Bootstrap**.

Visit the live project:  
https://btekhne.netlify.app/
